# 03_Career コンテキスト

## 役割
転職活動の全情報と、中長期キャリア戦略の管理。現在最優先フォルダ。

## オーナー概要
- バックグラウンド：IT/SaaS業界、BtoB営業・CS
- 転職ターゲット：SaaS企業のSales / CS / RevOps ポジション
- 現在フェーズ：書類選考〜面接（複数社並行）

## フォルダ構成

| フォルダ | 内容 |
|---|---|
| `10_プロフィール/` | 経歴・職歴・強み・実績の正本（事実ベース） |
| `20_マスタ/` | 履歴書・職務経歴書のマスタードラフト（汎用） |
| `30_キャリア戦略/` | 中長期キャリア方針・市場調査・スキルマップ |
| `40_転職活動/` | 今回の転職台帳・企業別進捗・面接Q&A |
| `50_自動化/` | 書類カスタマイズ・面接準備の自動化設計 |

## このフォルダでやること・やらないこと
- やること：実績の正本管理、書類作成、面接準備、企業調査、意思決定記録
- やらないこと：本業実務ナレッジ（→`01_Work/`）、AI戦略設計（→`04_AI/`）

## 主要ファイルの正本定義

| データ種別 | 正本の場所 |
|---|---|
| 定量実績（数字・期間・役割） | `10_プロフィール/` |
| 提出書類（汎用） | `20_マスタ/` |
| 企業別カスタマイズ書類 | `40_転職活動/企業名/` |
| 面接Q&A | `40_転職活動/面接Q&A.md` |
| 市場調査・キャリア方針 | `30_キャリア戦略/` |

## Claude Codeへの指示方針
- 書類生成時は `10_プロフィール/` の実績データを必ず参照すること
- 志望動機・強みのカスタマイズは企業情報と `30_キャリア戦略/` の軸を照合すること
- 個人情報・具体的な会社名は外部送信しないこと
- 数字は誇張せず、`10_プロフィール/` の記載値を使うこと
