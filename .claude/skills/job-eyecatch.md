# /job-eyecatch — 求人・アイキャッチ案

## これは何か

採用サイトや SNS 用の**バナー・サムネ**向けに、複数パターンの**画像プロンプト**と構図のメモを出すスキルです。媒体ごとのサイズも意識します。

## いつ使うか

- 新しい職種の**ビジュアルの方向性**を早く固めたいとき
- デザイナーに渡す前の**ブリーフ**が欲しいとき

## 使わないほうがよい場合

- **長文記事の章ごと画像** → `generate-article-images`
- ブランドのテンプレが固定で、差し替え不要のとき

## 前提条件

- 生成 API は任意。無ければプロンプトとレイアウト指示のみ。

## どう依頼するか

1. **`job-eyecatch`** と職種、一言の訴求、雰囲気。
2. 必要なピクセル（例: 1200×630 と正方形）。
3. 採用広告として避けたい表現（年齢・性別に偏る表現など）。

**例:** 「`job-eyecatch`。シニア CS。訴求は裁量と成長。横 1200×630 と正方形。年齢・性別で選別しているように見えない表現で。」

## 進め方（エージェント向け）

1. 想定する候補者像とメッセージの軸を確認する。
2. **3〜5 案**、構図が被らないようにする。
3. 上に文字を載せるなら**余白（セーフエリア）**をコメントする。
4. 実在のロゴや特定個人のそっくり表現は避ける。

## 完了の目安

- サイズごとに**使い分けの指示**がある
- 採用コンプライアンス的に危ない表現を避けている
- 法務・ブランドの最終確認が必要と明記されている
