# /generate-proposal-excel — 11シート想定の提案書 Excel

## これは何か

ヒアリングメモから、**提案書用の `.xlsx`（だいたい 11 シート：サマリーから次回アジェンダまで）**の構造と仮の数値を入れた草案、あるいは生成スクリプトの案を出します。**金額や工数は仮・要確認**です。

## いつ使うか

- 提案書の**型を毎回そろえたい**とき
- 相手が表で**内訳をいじりたい**タイプのとき

## 使わないほうがよい場合

- 納品がスライド中心 → `digirise-presentation`
- 人事評価のブック → `generate-eval-excel`

## 前提条件

- Python と openpyxl などが使えるか、または「シート設計だけ」かを決める。大きいブックは**小さく検証**から。

## どう依頼するか

1. **`generate-proposal-excel`** とヒアリングメモ・制約・見積の前提。
2. 既存テンプレのパス。通貨・税・端数のルール。
3. スクリプトで生成したいか、設計だけか。

**例:** 「`generate-proposal-excel`。メモ貼る。税込・端数は切り上げ。テンプレなし新規。Python での生成案。」

## 進め方（エージェント向け）

1. シートごとの**役割と主な列**を表で確定する。
2. 数式は**一方通行**で、循環参照を避ける。
3. まずサマリーと費用だけ、など**段階的に**試せる形を提案してよい。
4. 仮の数値には「要確認」とラベルを付ける。

## 完了の目安

- シート一覧が**レビューしやすい表**になっている
- 仮データが**本番と誤認されない**
- 印刷・列幅・ヘッダ固定など、確認項目が列挙されている
