    # /generate-proposal-excel — 11シート提案書Excel

    **やること:** ヒアリング内容から提案書 Excel（11シート構成）を生成

    **依存:** openpyxl 等の Python 環境、またはユーザーが Excel を編集

    ## 実行手順
    1. ヒアリングメモ・制約・見積条件を収集。
2. 完全まとめのシート一覧（サマリー〜次回アジェンダ）に従い、各シートの列・行構造を決める。
3. スクリプトまたはテンプレから .xlsx を生成。大型変更は段階的に検証。
4. 数値・工数は**仮**と明記し、ユーザーが最終確認する。

    ## Vault での共通ルール
    - クライアント固有情報・実名は扱わない（匿名化）。
    - `clients/` 等の例パスは、このリポジトリの慣習に**置換**する。議事録・アウトプットの置き場はユーザーに確認する。
    - MCP / API が無い操作は**テキスト案・手順だけ**出し、実行はユーザーが行う。
