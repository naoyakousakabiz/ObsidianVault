# /edit-vertical-video — 縦動画（ショート）の編集

## これは何か

TikTok / Shorts / Reels 向けの**縦動画**について、字幕・無音のカット・BGM のミックスなどを **`ffmpeg` を中心**にどう組むか、フィルタとコマンド例を段階的に提案するスキルです。

## いつ使うか

- 尺を詰めたり、無音部分を抜いたりしたいとき
- 字幕を焼き込むか、別ファイルにするか決めたいとき
- GUI ではなく、**コマンドで再現したい**とき

## 使わないほうがよい場合

- 細かいタイムライン操作がメインで、どうしても NLE 前提のとき
- **コードベースで説明動画の型**を作りたい → `remotion-video`

## 前提条件

- `ffmpeg` が使えること。試すときは `-ss` / `-t` で**短い区間**から試すのがおすすめ。

## どう依頼するか

1. **`edit-vertical-video`** と入力ファイルのパス、希望の解像度・fps。
2. 字幕ファイル（SRT など）の有無と、BGM の有無・目標のラウドネス。
3. 変えてはいけない条件（顔の画角はそのまま、など）。

**例:** 「`edit-vertical-video`。`clip.mov` を 1080×1920、30fps。SRT あり。無音 0.3 秒以上はカット。BGM は原音より下げめで。」

## 進め方（エージェント向け）

1. メタデータ（解像度・長さ）を確認する前提を置く。
2. **scale → crop → subtitle → 音量**など、フィルタを段階的に書く。
3. 1 行に全部詰めず、**試しやすい複数コマンド**に分ける。
4. 権利（素材・顔）とプラットフォームのルールは、ユーザー確認を促す。

## 完了の目安

- コマンドが**そのまま試せる**か、プレースホルダだけ置き換えれば試せる
- 字幕と音声の意図がパラメータに反映されている
- 大容量ファイル時のディスク容量に一言あるとよい
