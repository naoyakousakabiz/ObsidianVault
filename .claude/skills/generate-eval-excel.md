# /generate-eval-excel — 人事評価 Excel（15シート型）

## これは何か

職級・評価項目・配点のルールから、**15 シート想定の評価ブック**を設計し、`xlsx` の生成案か仕様書を出します。式やセル保護は**少しずつ**足していく前提です。

## いつ使うか

- 制度を変えて**テンプレのたたき台**が欲しいとき
- 本人・上司・HR など**入力者を分けたい**とき

## 使わないほうがよい場合

- 商談・提案の見積が主 → `generate-proposal-excel`
- 制度の**法的・労務の確定**だけが目的（ここでは下書きに留まる）

## 前提条件

- Python と openpyxl など、生成に使う道具を決めておく。

## どう依頼するか

1. **`generate-eval-excel`** と職級・軸・配点・半期か年次か。
2. 誰がどのシートを触るか。既存テンプレのパス。
3. 個人名より ID で扱いたいか。

**例:** 「`generate-eval-excel`。職級 3 段。軸は 5 つ。上司入力と本人入力を分ける。15 シート型で。」

## 進め方（エージェント向け）

1. 15 シートの**役割**と、入力→集計→確定の流れを表にする。
2. 集計はピボットに頼りすぎず、参照関係が追えるようにする。
3. 入力検証（プルダウンなど）を先に。式は最小から。
4. 法務・人事の正式確認が必要と必ず書く。

## 完了の目安

- シート間の**データの流れ**が説明できる
- 個人情報の扱いに**方針**がある
- 本番運用前の承認が必要と明記されている
