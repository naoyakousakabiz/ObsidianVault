## アイアンマン OWS 3.8km：クロールで完泳するためのノウハウ

> **目的**：スイムで「燃え尽きず／迷わず／怪我なく」3800mを終え、**ランに使える体**で T1 に入る。  
> **前提**：本編の **浮く・呼吸・フラット・長距離効率（14・15章・2ビート）** が土台。OWS はその上に **環境・コース・集団** が乗る。

### 勝ち筋（完泳最適化）

- **ペース**はプールのベストタイム追求ではなく、**バイク開始時に心拍と筋肉が回復しているライン**を守る。
- **エネルギー配分**：最初の 500m で狂うと後半が雪だるま。**テンションは低く始め、呼吸とリズムが安定してから**「レースペース思考」へ。
- **本編との接続**：足が沈む・息が上がるほど T1 以降が辛くなる → **第1〜3章（浮く・姿勢・呼吸）** と **第14〜15章（長距離・2ビート）** を最優先で OW に持ち込む。

### 3.8km の捉え方（距離感）

- プール換算の感覚を持つ：**約 76 本（50m）／約 152 本（25m）**。単純に「長いイージーセット」と割り切り、**分割**する。
- **分割例（イメージ）**：500m × 7 + 300m、または「ブイの区切り／ランドマーク」ごと。頭の中で **カウントダウンではなく区切り**（「ここまでは形」「ここからは維持」）の方が崩れにくい。
- **心拍**：OWS は波・緊張・集団で **陸上より上がりやすい**。序盤からレッドゾーンに入らない。

### オープンウォーターで追加で必要なスキル（本編だけでは足りない部分）

### レース当日の流れ（OWS 3.8km 用・ざっくり）

1. **ウォームアップ**：肺を広げる程度。ジップアップ前に肩・首を。**いきなり全力はしない**。
2. **スタート〜最初の 5〜10 分**：ハートレートが落ち着くまで **力を抜き**、**呼吸の3リズム**（第3章）を最優先。位置取りの修正は**足で焦らず**サイトイングで。
3. **中盤（メイン）**：**2ビート／軽いキック**（第12・15章）で脚と心拍を節約。浮力はウェット＋第1〜2章の意識。
4. **終盤〜アウト**：最後 200〜400m でスプリントしない（**ラン用に残す**）。岸・ランプが見えたらテンション管理に注意（転倒・ふらつき防止）。
5. **T1 心構え**：立ち上がりで急に走らず、**ふくらはぎが回るまで**移動。本編のフォームを**途中で捨ててゴールしたくなりやすい**区間なので、意識だけ先に持っておく。

## 1. 水泳の絶対原則：浮くこと

クロールが上達しない理由の大半は「浮けていないこと」に起因する。手の動かし方より先に解決すべき問題。

**なぜ浮くことがすべてか**

水の抵抗は空気の約800倍。足が10cm沈むだけで全身にブレーキがかかる。逆に水面に平行に浮ければ、少ない力でスーっと前に進む。ボートが浮いていれば少し漕ぐだけで進む、あの原理。

**70%の完成度から始める**

完璧を目指すと速さを求めてしまい、力で泳ぐ悪癖がつく。「浮いて手を回しているだけ」でよい。浮くことを徹底するだけでフォームは後から自然と整う。1%ずつ上げていく意識が正しい。

**浮力のメカニズム：重心と浮心**

- 人間の体は構造上、足から沈む。
- **浮心**（肺のあたり）が**重心**より上にあるとバランスが保たれる。
- 目線を下げる・手を前に伸ばす・腹に力を入れる、これらすべてが重心を前に押し出す行為。

## 2. 姿勢とフラットポジション

### 2-1. フラットポジションとは

水面に平行に体を保つ姿勢。すべての技術の前提。足が沈んでいる = ブレーキ状態。フラットになっている = 抵抗ほぼゼロ。

速度が上がるほど抵抗は掛け算的に増える。だからプロほど姿勢にシビア。

### 2-2. フラットポジションを作る3つのアクション

**① 目線を真下に向ける**  
目線が上がるほど足が沈む。「90°お辞儀」のような真下向きが理想（詳細は10章）。

**② 手を前に伸ばす**  
前に手があると体重が前にかかり足が浮く。手が空中にある時間が長いほど足が沈む。

**③ 腹（体幹）に力を入れる**  
みぞおちの少し上を膨らませ、わき腹をへこませる。腰の骨の内側がパンと張る感覚（詳細は8章）。

## 3. 呼吸法：クロールの9割を決める

**断言：25mで息が上がるのは力の問題ではなく呼吸の問題。**

### 3-1. 水中と陸上では呼吸が逆

| | 陸上 | 水中（クロール） |
|---|---|---|
| **吸う** | 鼻から | **口から** |
| **吐く** | 口から | **鼻から** |

水圧は1気圧以上かかっている。自分で「ふんっ」と押し出さないと空気は出ない。

### 3-2. 水泳の3リズム：吐く → 吸う → 止める

```
鼻からふんっ（吐く）→ 口からすっ（吸う）→ 止める　止める　止める…
```

**止める時間が最も長い。** 止めることで体に空気が残り浮力が保たれる。止める時間が長い人 = 上級者。

### 3-3. 呼吸の比率と段階的な目標

| ステージ | 吐く | 吸う | 止める |
|---|---|---|---|
| **まず目指す** | 3 | 3 | 4 |
| **中級の目標** | 2 | 2 | 6 |
| **上級者の実態** | 0.5 | 0.5 | 9 |

全部吐ききらなくていい。6〜7割吐ければ十分。全部吐くと沈みすぎる。

### 3-4. 吐く・吸う・止めるのタイミング

- **吐く：** 手が6時〜7時半でグッと押すとき（またはその直前）から鼻でふんっ
- **吸う：** 手が水から出た瞬間に口で。意識しなくても反射的に吸える
- **止める：** 手を前に伸ばしている間ずっと。この時間を長くするほど楽になる

**覚え方：** 花粉症で鼻をかむ「ふんっ」の感覚。これをやれば自然と吸える。吸う練習は不要、徹底的に吐く練習をする。

### 3-5. 3つのNG

**NG① 吐けていない（最頻出）**  
「ちゃんとやっている」と思っていても実際は細くしか出ていないことがほとんど。25mで息が上がるなら、まずここを疑う。

**NG② 止めていない**  
吐く→吸うを繰り返すだけで止めないと体が激しく上下して体力を消耗する。

**NG③ リズムがバラバラ**  
最初から最後まで同じリズム。疲れても速くなっても変えない。

## 4. 入水（エントリー）：ブレーキを最小化する

クロールにおいて**唯一ブレーキがかかる動作**。ここの質が上級者との最大の差。

### 4-1. 正しい入水の順番（絶対ルール）

```
指先 → 手首 → 肘
```

この順番で入れることで抵抗を最小化できる。肘から入るとバシャンと大きなブレーキになる。

- **入水位置：** 頭の前10〜15cm
- **入水の速度：** 途中で止まらない。豆腐を包丁でスッと切るように素早く入れる。水中で伸ばす

### 4-2. 途中で止めてはいけない理由

手が空中に出ている時間は体が沈む時間。水中での体重は1/10になるが手の重さは変わらない。手が空中にある = その重さで体が引っ張られて沈む。リカバリーを最短で行う理由もここにある。

### 4-3. 最大NG：肘落ち（大阪のおばちゃん）

肘と指先が同時に落ちる「ちょっとあんた」の動き。バシャンという音とともにブレーキが発生する。

**原因：** 手を遠くに伸ばそうとしすぎること、呼吸後に気が緩むこと。  
**改善：** 肘はギリギリまで上げたままキープ。ローリングを活用すれば自然とキープできる（7章参照）。

## 5. 入水の肘角度：プロとアマを分ける30°

### 5-1. 理想の入水角度

| 角度 | 状態 | 問題 |
|---|---|---|
| 90°（垂直） | 手のひらが正面から当たる | ブレーキ最大 |
| 5°（ほぼ水平） | 肘が落ちやすい | ブレーキ大 |
| **約30°** | **スーっと入る** | **ブレーキ最小** |

**イメージ：** 飛行機の着陸。ガンと急降下せず、スーっとなだらかに滑走路に入っていく感じ。

### 5-2. 肘の出っ張りの向きが命

肘の出っ張りは常に**斜め上か横向き**にキープ。下を向いた瞬間に肘は落ちてバシャンとなる。入水の前後を通じてこの向きを維持する。

かくときも同様。肘の出っ張りが下を向くと水をかけなくなる。横向きにすることで水をしっかりとらえる。

### 5-3. 入水の速度は一定に

うまい人は後ろから前に持ってくる速度と入水の速度が一定。途中で止まると体が沈み、そこからバシャンと落ちる原因になる。

### 5-4. 手の回し方の2段階構造

手を回す動きは「**ローリング**」と「**手の上下**」の2つの組み合わせ。ローリングしながら下げていくとスムーズな入水になる。「回す」意識より「ローリングしてから下げる」というイメージで調整しやすくなる。

## 6. 水中のかき方：5つの動作と力配分

### 6-1. 手の一回転を5つに分解する

| # | 動作 | 内容 | 力 |
|---|---|---|---|
| 1 | **エントリー** | 指先→手首→肘の順でスッと入れる | なし |
| 2 | **キャッチ** | 小指から引っかけて水の玉をとらえる | 弱 |
| 3 | **プル** | 6時〜7時半でグッと押す。唯一の強力ゾーン | **強** |
| 4 | **プッシュ** | 後ろに押し出す。上に上げない。足方向に向けて押す | 中 |
| 5 | **リカバリー** | 肩甲骨を引きながら最短で前へ | なし |

### 6-2. 手のひらの向き

手のひらは常に**足方向（後ろ）を向ける**。これができていないと水をかいても進まない。

- キャッチ後：なるべく早く後ろに向ける
- プルからプッシュ：ずっと後ろ（足方向）を向け続ける
- 手首が上を向いたらNG

**大きなボールに乗っていくイメージ**で手のひらが自然に下を向きながら前に出す。そこから後ろに押す。

### 6-3. 力を入れるのは6時〜7時半のみ

それ以外はすべて脱力。最初から力を入れると体が反って足が沈む。キャッチまでは「水を集めているだけ」。

**プールサイドに上がる動作を横向きにしたイメージ**が分かりやすい。

### 6-4. おへその近くを通す（上級テクニック）

手をおへそに近いところでかくほど力が出る。遠いほど力が弱くなる。プールサイドに上がる時に腕が体の近くにある方が力が出る、あの原理と同じ。

- 入水後、**小指側から**かき始めておへそに近づけていく
- おへそ付近で一瞬力を抜いて**親指に持ち替え**て押す
- 最後は手首だけひねって最後まで水を押しきる
- 体の中心線を超えて内側に入りすぎない（ブレーキになる）

### 6-5. 指の使い方

ぎゅっと握らず、かといって開きすぎない。軽く自然に閉じた状態。指先（小指・薬指側）に少し力を入れると水を引っかけやすい。

### 6-6. リカバリーは最短・脱力

「将棋の大手」「ポケットから手を抜く」感覚でスッと出す。よいしょと回してはいけない。ここで力を入れると手が遅くなり、体が沈み、リズムが崩れる。

## 7. ローリング：回転を推進力に変える

### 7-1. ローリングとは

クロールを泳ぐとき、体は「焼き鳥の串が刺さってくるくる回る」ように動いている。真ん中に軸があって、そこを中心に左右交互に傾く。軸はブレない。

### 7-2. ローリングがなぜ必要か

| 効果 | 内容 |
|---|---|
| 肘をキープしやすい | 体を横向けると手を上げるのが楽になる |
| 手の回転をコントロールできる | 回転スピードの調整が可能になる |
| 手が伸びやすくなる | ローリング＋肩甲骨の組み合わせで前に伸びる |
| 呼吸が取りやすくなる | 体が横向くと自然に呼吸できる |
| 体重移動ができる | ローリングが推進力になる |

### 7-3. こんにちはローリング

コースロープに向かって「**こんにちは**」と胸から挨拶するように、毎回しっかり横を向く。これができていればローリングは十分。

### 7-4. コツ：腰から・入水してから

- **肩から回してはいけない：** 肩だけが回り手が内側に入ってしまう
- **正しい順序：** 手が入水してから → 腰をグッとひねる → 体が自然に横向く → 手が伸びる
- **コースロープ側に腰をひねる**イメージが正確

## 8. 体幹：泳ぎの土台

### 8-1. なぜ水泳に体幹が必要か

陸上では地面が支えになる。水中には支えるものが何もない。体幹がふにゃふにゃだと、手で水をかく力が体に吸収されてしまい推進力にならない。体幹がしっかりしていると、手のかきが力として水に伝わる。

### 8-2. 体幹の作り方

みぞおちの少し上を膨らませ、わき腹をへこませる。腰の骨の内側がパンと張るような感覚でキープ。

呼吸で吐く瞬間はどうしても体幹が抜ける。止める時間中にしっかりキープする。止める時間が長いほどキープしやすい。

### 8-3. 手の伸ばしと体幹はセット

「ハチーっ」や「100万円！」と唱えながら前に手を伸ばすと体幹が入りやすい。指を1本1本広げるように伸ばすとさらに安定する。

## 9. 肩甲骨：上級者が必ず使う隠し武器

### 9-1. 肩甲骨を使う2つのシーン

**シーン①：前に伸ばすとき（かくとき）**  
ローリングした後、さらに肩甲骨を前に押し出す。これによって手が「伸びているように見える」。実際には手の関節は伸びない。**伸びているように見える = ローリング＋肩甲骨の組み合わせ**。

**シーン②：前に持ってくるとき（リカバリー）**  
「回す」のではなく「後ろの肩甲骨をグッと引いて前に持ってくる」。力を使わずスッと最短で手を前に持ってこれる。体が沈みにくくなる。

### 9-2. なぜリカバリーで肩甲骨を使うと沈まないか

よいしょとゆっくり持ってくると、その分だけ体が沈む。肩甲骨を引いてすっと持ってくれば、沈む前に手が戻ってくる。上級者が楽そうに見える理由の一つがここ。

### 9-3. 肩甲骨ウォームアップ（陸上）

肩をゆっくり前回り・後ろ回し、両肩を反対方向にクロスして回す。毎日続けると可動域が広がる。

## 10. 目線：ポジションを決める

### 10-1. 目線と足は直結している

| 目線 | 足の状態 |
|---|---|
| 上（前）を向く | 足が沈む |
| 真下を向く | 足が浮く |

泳ぎながら頭を上げると、一瞬で足が沈むのがわかる。

### 10-2. 正しい目線

- **通常時：** 真下（90°お辞儀のイメージ）
- **呼吸時：** 真横（コースロープが見えるくらい）

真下に向いた状態からそのまま体ごと横向きで呼吸。頭を上げない。

### 10-3. よくある失敗：歌舞伎役者呼吸

呼吸後に「いよーっ」と前を向いてしまう動き。頭の重さで体が立ち上がり足が沈む。

**改善：** 顎を上げてから横向く → コースロープが見えたらすぐ下向く。

## 11. 手のリズム：長距離と短距離を分けるタイミング

### 11-1. 短距離と長距離でリズムが違う

| タイプ | タイミング | 特徴 |
|---|---|---|
| **短距離** | 手が出た瞬間に反対の手を動かす | 回転速い・疲れやすい |
| **長距離** | **反対の手が入水してから動かす** | ゆっくり・疲れにくい |

### 11-2. 手をずらす：休憩できるクロールの核心

長距離向けの泳ぎの本質。**反対の手が入水する瞬間に今の手を動かす**というタイミング。

| 効果 | 説明 |
|---|---|
| 伸びる時間が生まれる | この間が「休憩」になる |
| 力が自然に抜ける | 余裕ができて脱力できる |
| 呼吸リズムが整う | タイミングに余裕ができる |
| 水をしっかりとらえる | 泡が取れてからかけるようになる |

**タイミングのコントロール：**

```
遠い（短距離的）：手が出た瞬間に動かす
        ↓
中間（長距離向け）：手が入水するころに動かす　← ここを目指す
        ↓
近い（練習用）：完全に前で揃えてから動かす（キャッチアップ）
```

### 11-3. 呼吸時に手が早くなる原因

呼吸のときに無意識で「よいしょ」と反対の手を早く動かしてしまうことが最多の原因。**顔を先に戻してから手を動かす**。顔と手が同時に動く = 強制的に短距離のリズムになる。

## 12. キック：最小限で最大の効果を出す

### 12-1. キックの正しい役割

| よくある誤解 | 実態 |
|---|---|
| 「キックで前に進む」 | 推進力への貢献は**20〜30%**。残りは手 |
| 「強く打つほどいい」 | 強く打つほど疲れる |
| 「足が沈んだら打つ」 | 常に軽く打ち続けて下半身を浮かせる |

**キックの主な役割 = 体を浮かせること・バランスを取ること。**

### 12-2. 正しいキックの3原則

**① 蹴ったら上げる**  
蹴り下げたままにしない。蹴ったらすぐ上げる。フラダンスのようにしなやかに止まらず動かし続ける。

**② 内股を意識する**  
まっすぐ蹴るより少し内股に蹴ることで可動域が広がり推進力が増す。プロペラのように外→内の回転力（揚力）を使う。親指同士が当たるイメージ。

**③ 付け根（太もも）から動かす**  
膝から下だけで蹴る「膝下キック」は進まない。付け根から動かすと膝から下がしなって自然と綺麗なキックになる。

**強さの目安：** 幼稚園の子が肩たたきするくらいの軽さ。トントントントン、日の2拍子。

### 12-3. 手・足・呼吸の合わせ技（上級）

右手が6時〜7時半でグッと押すとき → 右足でドン → 鼻からふんっ（同時）

この3つを合わせると推進力が2倍以上になる（1+1+1=5の合わせ技）。

## 13. 体重移動：最新クロールの核心

### 13-1. 昔のクロールと今のクロールの違い

| | 昔のクロール | 最新のクロール |
|---|---|---|
| スタイル | 体が少し反る、力で進む | フラット姿勢、体重移動で進む |
| イメージ | ぐーぐーとかく | スケートでスーっと滑る |
| 重点 | 手のかく力 | 水の抵抗を減らす姿勢 |
| 手の引き方 | 最後まで押す | **早めに抜いて前に体重を乗せる** |

### 13-2. スケートのイメージ

右手を入れるとき → 右半身ごと前に出す。左手を入れるとき → 左半身ごと前に出す。体を縦半分に切って左右交互に体重を乗せ換える。これがスーっと進む感覚の正体。

### 13-3. 手を早めに抜く理由

最後まで押しきろうとすると体が反って足が沈む。6時〜7時半でグッと押したらスッと前に手を戻す。足が沈む前に体重を前に乗せることでフラットポジションを保てる。

最初の練習では6時〜6時半で抜いても十分。体重が乗りやすくなる感覚を優先する。

## 14. 長く泳ぐための統合テクニック

### 14-1. 1000m泳ぐための3つの神ポイント

1. **呼吸比率334を厳守する** — 最初から最後まで同じリズム。崩れた瞬間から疲れが始まる
2. **手は入ってから動かすリズム** — 長距離用のゆっくりした回転。短距離のリズムで1000m泳ぐのは100mのリズムでマラソンするようなもの
3. **こんにちはローリング** — 毎回コースロープに向かって胸をしっかり向ける。手の伸び・呼吸・リズムがすべて安定する

### 14-2. 伸ばす時間を作る：ハチーの呪文

呼吸後にすぐ手を動かさず、心の中で「**ハチーっ**」または「**100万円！**」と唱えながら前に手を伸ばす。

この待つ時間が：
- ローリングと肩甲骨の伸びを生む
- 力が抜ける
- 体が浮く
- 呼吸のリズムが整う

### 14-3. 泳ぎながら休憩する方法

上級者が「優雅に見える」のは、伸ばしている時間に休憩しているから。**手をずらす**ことで休憩できる区間が生まれる。この区間に力を抜き、体幹を整え、次のかきに備える。

## 15. 2ビートクロール：長距離の最強テクニック

### 15-1. 2ビートクロールとは

手を左右1回ずつ回す間にキックを**2回だけ**打つ泳ぎ方。

| 特徴 | 詳細 |
|---|---|
| 向いている場面 | 200m以上の長距離、トライアスロン、オープンウォーター |
| 向かない場面 | 50mのスプリント |
| メリット | 疲れにくい、ある程度のスピードも出る |

### 15-2. キックのタイミング（これを間違えると進まない）

**右手でグッと押すとき → 右足でドン（同側）**

反対（クロス）はNG。理由：体重が乗りにくく、ローリングでの伸びが出ない。同側で蹴ると体が半分に切れたようにスーっと伸びる。スケートと同じ原理。

### 15-3. 同側とクロスの混乱について

「同側：右手を書くときに右足」と「クロス：右手が入るときに左足」は見かけ上のタイミングが同じになる。**「書くときに同じ足」**で覚えるのが最もシンプル。

### 15-4. 習得ステップ

```
Step 1: クロールドルフィン
        手はクロール・足はドーンドーンと2回のみ
        → タイミングを体に覚えさせる

Step 2: プルブイキック
        プルブイを挟んで体を浮かせ、タイミングだけ練習

Step 3: 片手クロール（2ビートバージョン）

Step 4: 通常の2ビートクロール
```

キックは強く打たなくていい。豆腐を優しく叩くくらいの力でタイミングを覚えることが先。

## 16. エラーチェックリスト

### 16-1. 手が下がる

| 原因 | チェック | 改善 |
|---|---|---|
| 入水角度が手前すぎる | 入水後にストンと真下に落ちる | 頭の10〜15cm前を狙って入れる |
| 肘落ち | バシャンと音がする | 肘の出っ張りを斜め上向きにキープ |
| 顔と手が同時に動く | 呼吸するたびに手が沈む | 顔を先に戻してから手を入れる |
| ローリングが肩から | 手が内側に入る | 入水後に腰からローリング |

**共通の改善：** 手が入ったら指先を上向きに。指を1本1本広げるイメージで前に伸ばすと下がりにくい。

### 16-2. すぐ疲れる

| 原因 | チェック | 改善 |
|---|---|---|
| 吐けていない | 25mで息が上がる | 鼻をかむ「ふんっ」を徹底 |
| 止めていない | ずっと吐いたり吸ったりしている | 止める時間を意識的に作る |
| 手が早く回りすぎる | 忙しそうに見える | 「入ってから動かす」リズムにする |
| キックを打ちすぎる | 足がバタバタ激しい | 軽く2拍子に変える |

### 16-3. 呼吸でスピードが落ちる

| 原因 | チェック | 改善 |
|---|---|---|
| 手が外に流れる | 呼吸側の手がスカスカになる | おへそ付近を通す意識 |
| 頭が上がりすぎる | 天井が見えるくらい顔が上がる | 真横を向く・顎を上げてから横向く |
| 呼吸後に前を向く | 歌舞伎役者呼吸 | 横を向いたらすぐ下向く |

### 16-4. 足が沈む

| 原因 | チェック | 改善 |
|---|---|---|
| 目線が上 | 前を向いている | 真下（90°お辞儀）を向く |
| 体幹が抜けている | 体がぐにゃぐにゃ | みぞおち上を膨らませる |
| 手を最後まで押す | プッシュで体が反る | 7時半でスッと前に抜く |

## 17. 練習メニュー完全リスト

### 17-1. 習得の優先順位と流れ

```
【基礎】呼吸の仕組みを体に刻む
  ボビング → 水中歩行 → サイドキック → 節付きキック
        ↓
【部分練習】1つの動作を集中して改善する
  ビート板片手クロール → フィンガーネイル → ショルダータッチ → 壁クロール
        ↓
【統合練習】複数の動作を組み合わせる
  キャッチアップクロール → 3回伸ばし → 片手クロール（両側）
        ↓
【応用練習】長距離・体重移動・2ビート
  スイッチ → プルブイキック → クロールドルフィン → 通常クロール
```

### 17-2. 練習メニュー詳細

| 練習名 | 目的 | 実施のポイント |
|---|---|---|
| **ボビング** | 呼吸の3リズムを体に刻む | 潜って止める → 鼻でふんっ → 上がって口で吸う。ゆっくり反復 |
| **水中歩行** | 呼吸タイミング・腕の動きを分解して練習 | 歩きながら腕を回す。苦しくなったら顔を上げられるので安全に反復できる |
| **サイドキック** | 呼吸タイミング・キックのバランス | 横向きでキックしながら呼吸。片側だけでなく必ず両側行う |
| **節付きキック** | フラットポジションの体感 | 手を前に伸ばしたままキック。背中が水面から少し出る感覚を覚える |
| **ビート板片手クロール** | 呼吸＋片方ずつのかき方 | 苦手側（利き呼吸の逆）を多めに練習。ゆっくり大きく |
| **フィンガーネイル（水面スレスレ）** | 肘落ち防止・入水角度の修正 | 指先で水面をなでながら前に運ぶ。頭の15cm前でストンと落とす |
| **ショルダータッチ** | 肘のキープ習慣 | 手を回すとき肩にタッチしてから前に持っていく。ゆっくり丁寧に |
| **壁クロール** | 呼吸と手のかきを合わせる | 壁を持った状態で手を回す。6時〜7時半で押すタイミングと呼吸を合わせる |
| **ドッグパドル風** | おへそ付近をかく軌道を習得 | 前向きで手の軌道を目で確認しながらかく。小指→抜き→親指の持ち替えを練習 |
| **キャッチアップクロール** | 手のリズム・手の伸ばし | 前で2秒止める（12と数える）。慣れたら徐々にタイミングを早めて通常クロールへ |
| **3回伸ばし** | ローリング・ハチーの練習 | 3ストロークごとにローリングしてこんにちは。1〜2秒キープしてから次へ |
| **片手クロール（両側必須）** | 入水・呼吸・タイミング全般 | 苦手側を多めに。両方バランスよくできると体がまっすぐ安定する |
| **スイッチ** | 体重移動の感覚を習得 | プルブイを挟んでグッと前に体重を乗せる動作の反復。前に乗る感覚を覚えることが目的 |
| **プルブイキック** | 2ビートのタイミング習得 | プルブイで体を浮かせてキックのタイミングだけ練習。強く打たなくてよい |
| **クロールドルフィン** | 2ビートのリズム習得 | 手はクロール・足はドーンドーンと2回のみ。軽く打つ |

### 17-3. 悩み別おすすめ練習セット

| 悩み | まずやること | 次にやること |
|---|---|---|
| 25mで息が上がる | ボビング → 水中歩行 | サイドキック → ビート板片手クロール |
| 手が下がる | フィンガーネイル → ショルダータッチ | キャッチアップクロール |
| 疲れやすい | キャッチアップクロール | 3回伸ばし → スイッチ |
| 長距離を泳ぎたい | キャッチアップクロール → 3回伸ばし | クロールドルフィン → 2ビートクロール |
| 泳ぎが進まない | 節付きキック（フラットポジション確認） | スイッチ → 片手クロール |
