# Claude Code スキル設計の要点

`.claude/skills/` のスキル（カスタムプロンプト）を書くときのチェックリスト。**Claude Code が意図どおり動くように、まず形を小さく固める**のが目的。

| ベストプラクティス | 理由 |
| --- | --- |
| 小さく始める | まず動くものを作り、後から改善 |
| `description` を具体的に | 自動マッチングの精度が上がる |
| 手順を番号付きで書く | Claude Code が順序を守りやすい |
| 出力フォーマットを指定 | 期待どおりの結果が得られる |
| エラーハンドリングを含める | 失敗時にスタックしない |
