# データ蓄積ガイド

各フォルダに蓄積すべきデータの一覧。「第二の脳」と「業務自動化」の両立を目的とする。

**原則：生ログではなく、再利用できる形に整えてから保存する。**

---

## 仕事軸

### 03_Career／キャリア戦略・転職管理

| データ種別 | 具体例 | 優先度 |
|---|---|---|
| 定量実績 | 数字・期間・役割・成果（KPI達成率、件数等） | ★★★ |
| スキルマップ | 現在値 vs 目標値のギャップ、習得ロードマップ | ★★★ |
| 面接Q&A蓄積 | 質問×回答×反省メモ（面接後即記録） | ★★★ |
| 市場情報 | 求人トレンド・給与相場・採用企業傾向 | ★★ |
| 企業情報 | 業界・事業・カルチャー・懸念点・決め手 | ★★ |
| 意思決定ログ | 転職・オファー承諾の判断根拠 | ★★ |
| 評価面談メモ | フィードバック内容・改善アクション | ★★ |

> 自動化活用：面接準備の自動生成、志望動機のカスタマイズ、書類の企業別派生

## 蓄積の優先順位（何から始めるか）

```
1. 03_Career：定量実績・面接Q&A  ← 転職活動中のため最優先
2. 03_Business/RINGBELL：対応パターン・テンプレ  ← 自動化の直接材料
3. 01_Work：Salesフレームワーク・CSナレッジ  ← 転職後に本格化
4. 09_Athlete：トレーニングログ・大会記録
5. それ以外：生活の質に関わるもの順で
```
