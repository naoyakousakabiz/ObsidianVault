# 日常AI自動化_実装指示書（最新版）

## 1. 実装スコープ

- P1-1 情報収集 → `#life-info`
- P1-2 朝ブリーフ → `#life-briefing`
- 食事/トレ通知 → `#life-reminder`
- P1-3 夜サマリ → `#life-summary`

## 2. チャンネルと環境変数

- `P11_SLACK_CHANNEL_ID`: P1-1（life-info）
- `P12_SLACK_CHANNEL_ID`: P1-2（life-briefing）
- `P13_SLACK_CHANNEL_ID`: P1-3（life-summary）
- `LIFE_OWNER_SLACK_ID`: メンション先ユーザーID
- `SLACK_BOT_TOKEN`: Slack Bot token（全ジョブ共通）

## 3. 実装仕様

### 3.1 P1-1（life-info）

- 実装: `scripts/p1-1-collect.py`
- 収集対象:
  - AIニュースTOP5（24h）
  - Web記事（24h・全件）
  - YouTube（24h・全件）
  - X（保留: API tier準備後）
- 通知要件:
  - 改行・区切りを入れた可読性重視
  - URLはタップ可能表示
  - `@owner` メンション付き

### 3.2 P1-2（life-briefing）

- 実装: `~/scripts/morning_briefing.sh`
- 方式: Webhook廃止、Bot tokenで `chat.postMessage`
- 表示項目:
  - 今日の予定
  - 今日の優先タスク（Daily抽出）
- 非表示項目:
  - 未返信メール候補
  - 情報収集ハイライト

### 3.3 life-reminder

- 実装: `~/scripts/life-reminder.sh <type>`（Vault正本: `scripts/life-reminder.sh`）
  - type: breakfast / lunch / snack / dinner / training
- チャンネル: `#life-reminder`（`LIFE_REMINDER_SLACK_CHANNEL_ID` = `C0ASQ7MTL7R`）
- 通知時刻（launchd）:
  - 朝食 07:30 → `com.lifeos.reminder-breakfast`
  - 昼食 12:00 → `com.lifeos.reminder-lunch`
  - 補食 15:30 → `com.lifeos.reminder-snack`
  - 夕食 19:00 → `com.lifeos.reminder-dinner`
  - トレーニング 06:00 → `com.lifeos.reminder-training`（時刻調整可）
- インストール: `bash scripts/install.sh reminder`

### 3.4 P1-3（life-summary）

- 実装: `~/scripts/life-summary.sh`（Vault正本: `scripts/life-summary.sh`）
- チャンネル: `#life-summary`（`LIFE_SUMMARY_SLACK_CHANNEL_ID` = `C0ATJHX5Z17`）
- 実行時刻: 21:00（`com.lifeos.life-summary`）
- 通知内容: Daily ノートから完了/進行中/未着手を集計 + 進捗バー + メモ抽出
- インストール: `bash scripts/install.sh summary`

1. `#life-info` へ `🧪 テスト配信`
2. `#life-briefing` へ `🧪 テスト配信`
3. `#life-reminder` へ `🧪 テスト配信`

確認観点:
- 送信主体がBotであること
- メンションが正しいこと
- 改行/装飾/URLリンクが崩れていないこと

## 6. 障害時フォールバック

- Slack送信失敗時:
  - ログを確認 (`~/scripts/*_error.log` または `90_運用ログ/`)
  - `SLACK_BOT_TOKEN` とチャンネル設定を確認
- 収集失敗時:
  - `p1-1-collector.log` を確認
  - 必要に応じて再実行
