# 05_Life コンテキスト

## 役割
人生設計・資産・住宅・保険・意思決定の管理。「生活インフラ」の正本フォルダ。

## フォルダ構成

| フォルダ | 内容 |
|---|---|
| `11_health/` | 健康管理（体重・測定・リマインド） |
| `20_events/` | ライフイベント計画（結婚・引越し・渡航のタイムライン） |
| `30_finance/` | 家計・資産概況・サブスク・FPライフプラン |
| `40_agreements/` | 同居ルール・インフラ契約・住戦略・二人会議 |
| `50_housing/` | 住宅（賃貸履歴・購入検討・ローンシミュレーション） |
| `60_insurance/` | 保険（種類・補償内容・見直しログ） |
| `70_decisions/` | 重要意思決定ログ（判断根拠・選択肢・結果） |
| `80_documents/` | 重要書類の在り処メモ（内容は別管理） |
| `99_archive/` | 完了したイベント（YYYYMMDD_イベント名.md形式） |

## このフォルダでやること・やらないこと
- やること：人生の重要な意思決定・資産・住宅・保険の管理
- やらないこと：キャリア戦略（→`03_Career/`）、趣味（→`09_Athlete/`・`10_Culture/`）、人間関係（→`08_Relationship/`）

## ファイル命名ルール
- イベント系：`YYYYMMDD_イベント名.md`
- 継続管理系：内容を表す名詞で命名

## Claude Codeへの指示方針
- 意思決定ログは判断根拠・選択肢・結果をセットで記録すること
- 書類の在り処メモには実際の内容（口座番号・証券番号等）を書かないこと
- 資産数値は概算レベルで記録し、詳細は別システムで管理すること
