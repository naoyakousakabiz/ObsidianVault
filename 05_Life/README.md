# 05_Life

**二人の共同生活の運用ハブ**（期限・家計・合意・健康管理）。キャリアや仕事の意思決定は `03_Career` / `01_Work`。ここは **Life** の語感どおり「人生全体」ではなく、**世帯として回すオペ**に寄せた置き場。

- **住まいの方針・外注・防災の考え方**など → [[40_agreements/README]]
- **二人の健診・検査・測定・予防まわり** → [[11_health/README]]

**個人を特定しやすい情報は書かない／伏せる**（Vaultルールに従う）。

## `20_events` と `40_agreements` の違い（目安）

| | **events** | **agreements**（継続テーマ・決まり事） |
|:--|:--|:--|
| 時間軸 | 開始・終了・**締切**が立つ | **継続的**。〆日がなくてよい（必要なら「見直し日」だけ） |
| 載せる内容 | 引越し、入籍手続、顔合わせ、撮影日が決まったら、など | 家事の分担**方針**、外注、住戦略、ふたり会議の蒸留、など |
| 片付き | 終わったら `99_archive` や日付ノートの完結でよい | ずっと生きる。更新・版メモが中心 |

迷ったら：**「カレンダーに1イベントとしてブロックを取るか？」** → Yes なら `20_events`、**常識・ルールとして残すか？** → Yes なら `40_agreements`。
