## 入籍まで（前日まで）

- [ ] **婚姻届**：提出先役所・受付時間、必要書類（戸籍謄本・住民票・印鑑等）、**証人2名**の確保（当日連れて行くか・事前手続の有無は管轄で要確認）

- [ ] **入籍日の勤務**：双方仕事の有無、休み・早退の取り方
- [ ] 入籍後の手続き（下記）を一度通読し、窓口の割り振り

## 入籍直後〜数週間（一般的・要確認）

※ 会社・保険・管轄で異なる。順不同。

- [ ] **会社への届出**：人事・総務（配偶者・扶養・住所変更・緊急連絡先など）
- [ ] **健康保険**：扶養の追加／外れる／第3号被保険者など
- [ ] **住民票・戸籍**：改製謄本が要るタイミングのメモ
- [ ] **マイナンバー・氏名変更**：入籍に伴う役所手続き（管轄確認）
- [ ] **年金**：第3号被保険者届等
- [ ] **銀行・カード・各種契約**：氏名変更の届出
- [ ] **パスポート**：氏名変更後の再取得（[[TBD_NY渡航と滞在]] の計画と整合）
- [ ] **各種保険の受益者**（生命保険等）
- [ ] **免許**等：氏名変更時の更新
