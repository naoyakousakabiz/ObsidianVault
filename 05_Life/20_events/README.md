# 20_events 運用ガイド

## 基本方針
- `20_events/` は **年別フォルダ** で管理する。
- 今年の正本は `2026/2026_ライフイベント・TODOまとめ.md`。

## 命名ルール
- 日付確定イベント: `YYYYMMDD_イベント名.md`
- 日付未確定イベント: `TBD_イベント名.md`

## 現在の構成
- `2026/`
  - `2026_ライフイベント・TODOまとめ.md`
  - `20260405_プロポーズ.md`
  - `20260429_ご両親挨拶.md`
  - `20260502_入籍.md`
  - `20260523_両家顔合わせ.md`
  - `20260613_引越し.md`
  - `TBD_*` 各イベントノート

## 運用メモ
- `TBD_` が確定したら `YYYYMMDD_` にリネームする。
- 年末に翌年分（例: `2027/`）を作成し、必要ノートを複製して引き継ぐ。
- 年別で保持するため、イベント終了後に `99_archive` へ移動しない。
