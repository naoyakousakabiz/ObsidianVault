# 01_Work — 本業実務・スキル蓄積

**役割**: 会社員としての日々の実務ナレッジ・スキル蓄積。  
キャリア戦略・転職管理は `03_Career/`。AI自動化設計は `04_AI/02_automation/`。

---

## フォルダ構成

- `10_スキル/` — 会社をまたいで持ち運べるスキルノウハウ
  - `sales/` — 商談管理・営業メソッド・交渉
  - `cs/` — オンボーディング・更新・ヘルス管理
  - `common/` — プレゼン・社内外調整・共通スキル
- `20_プレイブック/` — 業務フロー・テンプレート・チェックリスト
- `30_プロジェクト/` — 社内プロジェクト（匿名化・コード名で管理）
- `40_マネジメント/` — チーム・1on1管理（管理職になったら開設）
- `80_メモ/` — 会議メモ・1on1ログ（後で10か20へ昇格）

---

## 書き方の原則

- `80_メモ/` に一時保存 → 再利用できる知識だけ `10_スキル/` か `20_プレイブック/` へ昇格
- クライアント固有情報は入れない。パターン化した知識として `10_スキル/cs/` に書く
- 自動化設計は `04_AI/02_automation/` に置き、出力物をここに着地させる
