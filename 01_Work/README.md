# 01_Work — 本業実務・スキル蓄積

**役割**: 会社員としての日々の実務ナレッジ・スキル蓄積。  
キャリア戦略・転職管理は `03_Career/`。AI全体戦略・ツール設計は `04_AI/`。

---

## フォルダ構成

- `10_スキル/` — 会社をまたいで持ち運べるスキルノウハウ
  - `10_common/` — プレゼン・社内外調整・共通スキル
  - `20_CS業務/` — オンボーディング・更新・ヘルス管理
  - `30_営業/` — 商談管理・営業メソッド・交渉
- `20_プレイブック/` — 業務フロー・テンプレート・チェックリスト
- `30_プロジェクト/` — 社内プロジェクト（匿名化・コード名で管理）
- `40_マネジメント/` — チーム・1on1管理（管理職になったら開設）
- `50_AI自動化/` — 本業領域の業務自動化設計・実装（AI戦略は `04_AI/` が正本）
- `80_メモ/` — 会議メモ・1on1ログ（後で10か20へ昇格）

---

## 書き方の原則

- `80_メモ/` に一時保存 → 再利用できる知識だけ `10_スキル/` か `20_プレイブック/` へ昇格
- クライアント固有情報は入れない。パターン化した知識として `10_スキル/20_CS業務/` に書く
- 業務自動化の実装は `50_AI自動化/` に置く。AI戦略・設計思想は `04_AI/` を参照
